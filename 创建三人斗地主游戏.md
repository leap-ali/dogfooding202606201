给我创建一个3个人斗地主的扑克牌游戏，需要包含一个有前端页面web，有后端项目java。生成的项目文件，存放在：E:\数据标记\github的代码\git20260620的代码\solo1  这个目录下，具体需求如下： 
 1. 项目概述 
 项目名称：Web 三人斗地主在线对战平台 
 技术栈： 
 前端：Vue3 + Vite + Element Plus + Socket.IO + Pinia + SCSS 
 后端：Java SpringBoot 2.7 + MyBatis-Plus + Socket.IO Java + MySQL8.0 
 通信：WebSocket（实时同步发牌、出牌、抢地主、对局状态） 
 数据库：MySQL 存储用户、对局记录、积分排行 
 核心玩法：标准 3 人斗地主规则，游客 / 账号登录，在线匹配房间、实时对战、积分结算、历史对局查询。 
 2. 功能性需求 
 2.1 用户模块 
 用户注册、账号密码登录、游客快捷登录（无需注册） 
 用户信息：头像、昵称、积分、对局胜场 / 负场统计 
 登录认证：JWT token 鉴权，前端存储 token，接口拦截未登录请求 
 积分规则：地主赢 + 3 倍底分，农民赢各 + 1 倍底分；输对应扣除积分，积分低于 0 限制开局 
 2.2 房间匹配模块 
 大厅房间列表：显示房间人数（0/1/2/3）、底分、房间状态（空闲 / 对局中） 
 创建房间：自定义底分（1/2/5/10 分档） 
 加入房间：满 3 人自动开局，不足 3 人等待玩家进入 
 退出房间：未开局可自由退出；对局中退出判定逃跑，扣除双倍积分 
 2.3 斗地主核心对局业务（WebSocket 实时推送） 
 标准三人斗地主规则实现 
 发牌逻辑：54 张扑克牌，3 玩家各 17 张，底牌 3 张 
 抢地主环节：随机顺序抢地主，可抢 / 不抢，多次争抢，无人抢则重新发牌 
 地主拿 3 张底牌，手牌扩充至 20 张 
 出牌规则：单张、对子、三张、三带一、顺子、连对、飞机、炸弹、王炸；炸弹 / 王炸可压一切普通牌，王炸最大 
 轮流出牌逻辑：上一家出牌后下家必须出更大牌或选择不出 
 胜负判定：地主手牌出完→地主胜利；任意农民手牌出完→农民双人胜利 
 实时对局同步：所有玩家手牌、桌面出牌、操作状态实时推送到全部房间客户端 
 2.4 记录与排行模块 
 个人历史对局记录：对局时间、底分、身份（地主 / 农民）、胜负、积分变动 
 全局积分排行榜：展示前 20 名高积分用户 
 2.5 前端页面功能 
 登录页：账号登录 / 游客登录切换 
 游戏大厅页：房间列表、创建房间按钮、个人积分信息、排行榜入口 
 房间等待页：展示当前房间玩家、等待倒计时、退出房间按钮 
 对局游戏主页面： 
 自身手牌区域（可点击选牌、出牌按钮） 
 另外两名玩家手牌背面展示、剩余牌数 
 桌面出牌区域、底牌展示 
 操作按钮：抢地主 / 不抢、出牌、不出、提示、重置选牌 
 对局结算弹窗：展示胜负、积分增减、再来一局 / 返回大厅 
 3. 非功能性需求 
 实时性：WebSocket 毫秒级同步对局操作，无延迟 
 兼容性：Chrome/Edge 主流浏览器适配，移动端基础适配 
 稳定性：断线重连机制，断开后自动重连房间恢复对局状态 
 工程分层： 
 前端：视图层、状态管理层、接口请求层、socket 通信层、工具类拆分 
 后端：Controller 控制层、Service 业务层、Mapper 数据层、WebSocket 会话层、实体层、工具类 
 可启动性：前后端提供完整启动脚本、依赖配置、数据库初始化 SQL 
 4. 项目工程拆分 
 前端项目：ddz-web 独立 Vite 工程 
 后端项目：ddz-server SpringBoot Maven 工程 
 数据库：MySQL 初始化 ddz 库 + 表结构 SQL

## 核心功能

### ✅ 用户模块
- 账号注册/密码登录
- 游客快捷登录
- JWT Token 鉴权
- 积分、胜场/负场统计

### ✅ 房间匹配
- 大厅房间列表（人数、底分、状态）
- 创建房间（底分可选 1/2/5/10）
- 加入房间（满3人自动开局）
- 退出房间（对局中逃跑扣双倍积分）

### ✅ 斗地主核心玩法
- 54张牌发牌（各17张 + 3张底牌）
- 随机顺序抢地主，一轮制
- 牌型支持：单张、对子、三张、三带一、顺子、连对、飞机、炸弹、王炸
- 炸弹/王炸可压一切普通牌
- 轮流出牌 + 不出机制
- 胜负判定（地主出完 vs 农民出完）
- 智能提示功能

### ✅ 实时通信
- Socket.IO 毫秒级同步
- 断线自动重连
- 房间状态实时推送

### ✅ 记录排行
- 个人历史对局记录
- 全局积分排行榜（前20名）
- 积分结算：地主赢+3倍，农民赢各+1倍

## 启动方式

### 1. 数据库初始化
```bash
mysql -u root -p < sql/init.sql
```

### 2. 启动后端 (端口 8080 + 9092)
```bash
cd ddz-server
mvn spring-boot:run
```

### 3. 启动前端 (端口 5173)
```bash
cd ddz-web
npm install
npm run dev
```

访问 http://localhost:5173 即可开始游戏。

测试账号：`admin` / `admin123`（或直接使用游客登录）
